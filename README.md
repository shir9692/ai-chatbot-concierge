# AI Chatbot Concierge

IMT 598C ai-chatbot-concierge

## Overview

An AI-powered chatbot concierge system for hotel guests and staff.

## Quick Start

### Installation

```bash
npm install
npm start
```

The server will start and you can access the chatbot interface at `http://localhost:3000`.
